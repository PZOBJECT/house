# 🏠 出租房收租管理系统 — 设计系统

## 提取来源

| 文件 | 提取内容 |
|------|----------|
| `Dashboard.vue` | KPI 卡片渐变色、进度条、收入分解卡片 |
| `RoomManage.vue` | 房间表格、楼层筛选、出租统计条 |
| `BillManage.vue` | 已缴费/未缴费徽章、表格样式 |
| `App.vue` | 顶栏渐变、导航按钮 |

## 核心风格原则

1. **渐变为纲** — KPI 卡片大面积使用渐变，而非纯色
2. **元素加粗** — 重要数据 800/700 粗体，视觉冲击强
3. **圆角现代** — 卡片 14px、按钮 8px、徽章 20px 胶囊
4. **白色底色** — 列表/表格纯白底，与彩色 KPI 形成对比
5. **Element Plus 蓝色系** — `#409eff` 作为主色贯穿始终
