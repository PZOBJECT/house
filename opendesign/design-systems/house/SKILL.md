# 🏠 House Rental Management — Design System

## Overview
Design tokens extracted from the house project (Vue 3 + Element Plus rental property management app).

## Quick Reference

### Colors
- **Primary** `#409eff` — actions, links, active states
- **Success** `#67c23a` — paid, rented
- **Warning** `#e6a23c` — money values, rent
- **Danger** `#f56c6c` — unpaid badges

### Gradients (KPI Cards)
- Income: `#667eea → #764ba2`
- Paid: `#11998e → #38ef7d`
- Unpaid: `#f093fb → #f5576c`
- Occupancy: `#4facfe → #00f2fe`
- Header: `#409eff → #337ecc`

### Typography
- KPI values: 26px / 800 weight
- Room numbers: 18px / 700 weight
- Money values: 17px / 600 weight, #409eff color
- Labels: 13px / 500 weight

### Radius
- Cards: 14px
- Buttons/inputs: 8px
- Badges: 20px
- Floor cards: 10px

### Shadows
- Cards: `0 1px 4px rgba(0,0,0,0.04)`
- Card hover: `0 4px 16px rgba(0,0,0,0.08)`
- KPI hover: `0 6px 20px rgba(0,0,0,0.1)`
