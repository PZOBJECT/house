🔄 月度业务流转 每月一次
1 月初 — 生成当月账单 POST /api/bills/generate → 为所有已租房间创建当月账单 →
2 录入电表读数 读取每间房电表，填入"上月读数"自动带出，"本月读数"手动录入 →
3 录入水表读数 读取每间房水表，填入底数，自动计算用水量 →
4 自动计算费用 用电量×电费单价 + 用水量×水费单价 + 月租金 = 总费用 →
5 通知租客 / 催收 查看数据大屏未缴费列表，确认待催收房间 →
6 收取费用 → 标记已缴费 收到费用后，在系统标记该房间当月为已缴费（PUT /api/bills/:id/pay） →
7 月末 — 数据汇总 & 开收据 数据大屏展示月度汇总，已缴账单可生成收据图片下载 →
🌐 API 接口映射 前后端交互
接口 功能 关联页面
GET /api/rooms获取房间列表（支持出租状态筛选）房间管理、数据大屏
PUT /api/rooms/:room_no修改房间价格、出租状态房间管理
POST /api/bills/generate按年月生成当月所有已租房间账单月度账单
GET /api/bills/list获取账单列表（年月/房间/状态筛选）月度账单、历史账单
PUT /api/bills/:id修改水电读数（未缴费可改）月度账单
PUT /api/bills/:id/pay标记已缴费 ✅月度账单、数据大屏
PUT /api/bills/:id/unpay取消缴费 ❌月度账单
GET /api/bills/:id/receipt获取收据数据（用于截图下载）收据生成
